# Teste - Simulação de Crédito Imobiliário 

#  Objetivo
Este projeto tem como objetivo realizar testes manuais na jornada de **simulação de crédito imobiliário do Banco Santander**.
---

#  Banco Testado
**Santander Brasil**

# Jornada Testada
Simulação de Crédito Imobiliário (Pessoa Física)

---
# Tipo de Teste 
- Teste Manual – Jornada E2E
---

# Escopo dos Testes
- Acesso ao simulador de crédito imobiliário
- Preenchimento de dados da simulação
- Validações de campos obrigatórios
- Exibição de valores simulados
- Mensagens de erro para dados inválidos

---

# Ambiente de Teste
- Canal: Website Santander  
- Dispositivo: Desktop  
- Sistema Operacional: Windows 11
- Navegador: Google Chrome
